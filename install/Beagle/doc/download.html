download Beagle

Download Beagle, unpack the zip-file to a folder of your choice and run Beagle.exe, Beagle.bat or Beagle.sh to start the software. If Java is not installed, download the Java runtime environment (JRE) from here and run the installer.

java

The software runs on all platforms that support the Java Runtime Environment JRE 1.7 or higher. To test whether Java is installed run java -version. The console output should be similar to this: 

java version "1.7.0"
Java(TM) SE Runtime Environment (build 1.7.0-b147)
Java HotSpot(TM) 64-Bit Server VM (build 21.0-b17, mixed mode)
